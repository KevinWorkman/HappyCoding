This example uses [JSP](/tutorials/java-server/jsp) to show today's date:

```jsp
<%@ page import="java.util.Date" %>
<!DOCTYPE html>
<html>
  <head>
    <title>JSP Date</title>
  </head>
  <body>
    <h1>JSP Date</h1>
    <p>The current date is: <%= new Date() %></p>
  </body>
</html>
```
